Week 15

Week 15 Dinner Plan

周四 豆腐 + 猪肉 工作日 · 快手
肉末豆腐
简单儿童友好
猪肉末炒香 → 嫩豆腐下锅 → 生抽 + 少量老抽收汁，约 15 分钟
醋溜白菜
简单儿童友好
白菜手撕 → 蒜末爆香 → 大火爆炒 + 白醋，脆嫩下饭
紫菜虾皮蛋花汤
简单汤儿童友好
开水冲泡紫菜虾皮 → 打入鸡蛋搅散 → 加盐葱花，5 分钟搞定
备菜提示：豆腐可提前切好备好；白菜撕块备用，下班到家 20 分钟内全部上桌。
周五 鸡肉日 工作日 · 快手
可乐鸡翅
简单儿童友好可提前腌
鸡翅两面煎上色 → 倒可乐 + 生抽 → 中火焖 15 分钟收汁
清炒西蓝花
简单儿童友好
焯水 40 秒 → 蒜末爆香 → 大火翻炒 2 分钟，清脆不软烂
番茄蛋花汤
简单汤儿童友好
番茄炒出汤汁 → 加水烧开 → 打蛋花 + 盐，酸甜开胃
备菜提示：鸡翅周四晚用盐 + 生抽提前腌，冰箱隔夜入味更好；周五下班直接下锅。
周六 红烧 · 周末主菜 周末 · 中等难度
红烧排骨
中等儿童友好可提前炖
排骨焯水 → 炒糖色上色 → 加姜 + 生抽 + 老抽 → 小火焖 40 分钟收汁
蒜蓉炒时蔬
简单儿童友好
上海青 / 菠菜任选 → 蒜末 + 大火快炒 2 分钟，解腻
土豆炖排骨（延伸）
简单儿童友好
红烧排骨留一半 → 加土豆块 + 水继续炖 20 分钟，变成另一道菜
拍黄瓜
简单可提前备
黄瓜拍碎 → 蒜末 + 盐 + 香醋 + 少量香油，拌匀即可
周六主打一个「一次炖大锅排骨，变两道菜」：原味红烧排骨 + 土豆炖排骨，节省时间又不重复。
周日 周末压轴 · 鱼 + 虾 周末 · 可稍复杂
清蒸鲈鱼
简单儿童友好提前腌 15 分钟
鲈鱼划刀 → 姜丝 + 盐腌 → 上锅蒸 8 分钟 → 淋蒸鱼豉油 + 浇热油
蒜蓉粉丝虾
中等儿童友好
粉丝泡发铺底 → 虾开背去线摆放 → 蒜蓉 + 生抽调汁淋上 → 蒸 6 分钟 + 淋热油
麻婆豆腐
中等微辣可调
豆腐切块焯水 → 肉末炒香 + 豆瓣酱（少放）→ 加豆腐轻翻 → 水淀粉勾芡
番茄炒蛋
简单儿童友好
鸡蛋炒出锅备用 → 番茄炒软出汁 → 回锅合炒 + 少量糖提鲜
麻婆豆腐豆瓣酱量减半，微辣不辛，孩子接受度高；鱼和虾同步蒸，节省灶台时间。
本周食材复用一览
跨天复用
排骨（周六红烧 → 土豆炖） 豆腐（周四肉末豆腐 + 周日麻婆） 番茄（周五汤 + 周日炒蛋） 鸡蛋（周四汤 + 周日炒蛋） 猪肉末（周四 + 周日麻婆）
全周常备调料
葱姜蒜（每天用） 生抽 / 老抽 盐 / 白醋 / 香醋 淀粉（勾芡） 蒸鱼豉油
