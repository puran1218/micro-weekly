Week 16 Dinner Plan

第16周 · 清淡易消化，蒸煮为主，不放辣，孩子生病调养期。
周一 4月13日 清蒸 · 开周轻松 时间充裕
清蒸鲈鱼
简单儿童友好提前腌 15 分
姜丝铺底 → 蒸 8 分钟 → 淋蒸鱼豉油 + 浇热油，鱼肉嫩滑易消化
清炒上海青
简单儿童友好
蒜末爆香 → 大火翻炒 2 分钟 → 少许盐，清爽不油腻
鱼是生病期间最好的蛋白质来源，清蒸零难度，孩子也喜欢。
周二 4月14日 蒸蛋 · 快手 工作日 · 20 分钟
蒸水蛋
简单儿童友好
鸡蛋 + 1.5 倍温水打散过筛 → 盖保鲜膜蒸 10 分钟 → 淋生抽 + 香油
番茄炒蛋
简单儿童友好
鸡蛋先炒出锅 → 番茄炒软出汁 → 合炒 + 少许糖，酸甜开胃
冬瓜排骨汤
简单汤儿童友好可提前炖
排骨焯水 → 冷水入锅 + 姜片 → 烧开转小火炖 40 分钟 → 加冬瓜再炖 10 分钟，清甜去火
冬瓜清热利水，汤炖好后撇去浮油，给孩子喝汤就很好。可以周一晚上提前炖好。
周三 4月15日 豆腐 · 易消化 工作日 · 20 分钟
肉沫豆腐
简单儿童友好
猪肉沫炒香 → 嫩豆腐下锅 → 生抽 + 少量老抽轻翻收汁，不放豆瓣酱
醋溜白菜
简单儿童友好
白菜手撕 → 蒜末爆香 → 大火快炒 + 白醋，脆嫩清口
周二排骨汤如有剩余，今天热一热继续喝，不用另备汤。
周四 4月16日 鸡肉 · 补充体力 工作日 · 20 分钟
白切鸡
简单儿童友好可提前煮
鸡腿冷水入锅 + 姜片煮开 → 转小火 20 分钟 → 冰水过一下切片，蘸姜葱油
清炒西蓝花
简单儿童友好
焯水 40 秒 → 蒜末爆香 → 快炒 2 分钟，翠绿不软烂
鸡汤挂面（煮鸡余汤）
简单儿童友好
煮鸡的清汤留用 → 加盐调味 → 下细挂面煮熟，孩子生病最滋补的一碗
煮鸡余汤别倒，直接下面条，孩子吃一碗暖胃补体力，比任何药膳都实在。
周五 4月17日 蒸虾 · 换口味 工作日 · 20 分钟
蒜蓉蒸虾
简单儿童友好
虾开背去线 → 蒜蓉 + 少量生抽调汁淋上 → 蒸 5 分钟，鲜甜无腥味
清炒土豆丝
简单儿童友好
土豆切丝过凉水去淀粉 → 大火爆炒 + 少许白醋，清爽脆口
虾蛋白质高且易消化，孩子还在恢复期，蒸虾比炒肉更温和。
周六 4月18日 暖胃 · 收尾 周末 · 轻松
番茄炒蛋
简单儿童友好
经典做法，孩子爱吃，食欲恢复期最合适
清炒娃娃菜
简单儿童友好
娃娃菜手撕 → 蒜末爆香 + 快炒 2 分钟，清甜软嫩
豆腐蛋花汤
简单汤儿童友好
嫩豆腐切块入清水 → 烧开打蛋花 → 盐 + 葱花，10 分钟，清淡养胃
周六轻松，全部 20 分钟内完成。孩子若已好转，周日可以加回荤菜。
周日 4月19日 鸡汤 · 周末调养 周末 · 时间充裕
玉米山药鸡汤
简单汤儿童友好慢炖 1 小时
鸡块焯水 → 加玉米段 + 山药 + 姜片 → 炖 1 小时，汤清味甜，健脾补气
清蒸鸡蛋羹
简单儿童友好
2 个蛋 + 200ml 温水打散 → 过筛去泡 → 盖保鲜膜蒸 10 分钟，滑嫩零难度
清炒时蔬（任选）
简单儿童友好
上海青 / 菠菜 / 西蓝花任选一 → 蒜末爆香快炒，配汤喝刚好
山药健脾，玉米清甜，孩子快好了但肠胃还嫩，这锅汤是最好的过渡。
本周食材复用一览

鸡蛋（周二蒸蛋 + 周二炒蛋 + 周六炒蛋 + 周日蛋羹） 番茄（周二炒蛋 + 周六炒蛋） 嫩豆腐（周三肉沫豆腐 + 周六汤） 鸡汤（周四煮鸡余汤 → 直接下面条） 蒜末（全周通用）
